# Contributing

- Install Python 3.11 and create venv (see README).
- Activate pre-commit hooks: \pre-commit install\.
- Branch naming: \eat/...\, \ix/...\, \docs/...\.
- Commit style: Conventional Commits (e.g., \eat: add flood ETL\).
- Run \setup/refresh-map.ps1\ before pushing to keep demo up to date.
